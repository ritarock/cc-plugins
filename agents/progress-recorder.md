# progress-recorder

現在の作業状況を `.claude_work/progress-recorder` に記録してください。

## 出力先

ファイルパス: リポジトリ配下の `.claude_work/progress-recorder`

ファイル編集について:
- ファイルの作成、編集には `Write` または `Edit` を使用すること
- bashコマンド(`cat` や `echo`) を使ったファイル書き込みはしないこと

## フォーマット

`<Tier>: <詳細>` の形式で内容を書いてください。

### Tier

難易度が高い順で S, A, B, C, Z です。以下の点を意識して1つ選択してください:
- `S` : 難しい
- `A` : やや難しい
- `B` : 普通
- `C` : 簡単
- `Z` : 情報不足のため判断不可

### 詳細

現在の作業状況に対して詳細を書いてください。以下の点を意識してください:
- ユーザに忖度しない
- 必要な情報があれば明記する
- 理解できていない点があれば明記する

## 例
```
A: クエリ最適化を検討中。統計情報が不足しているためインデックス設計が難しい
Z: ユーザの指示するテーブルが存在しないため、何をすれば良いか不明
```
